# 논리 연산자 :
    1) and : A and B -> A와 B가 모두 참일 때 실행문 실행 &&
    2) or : A or B -> A혹은 B중에 하나가 참이라면 실행문 실행 ||
    3) not : if not A -> A가 false일 때 실행문 실행 !
# if else문
    if 조건문의 전체 형식 :
    if 조건식1 :
        실행문1
    (elif 조건식2 :)
        (실행문2)
    (elif 조건식3 :)
        (실행문3)
    (else:)
        (실행문)
    ()는 optional이다.
# while 반복문
    while 다음에 오는 조건식이 참이라면 이하의 실행문이 반복 실행됨(조건문이 False가 될 때까지)
    형식 :
    while 조건식1:
        반복실행문

    특정 시점에 while 반목문이 종료될 수 있도록 지정해야한다
